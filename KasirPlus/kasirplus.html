Kebijakan Privasi

Kasir Plus menjaga transparansi penuh tentang bagaimana data Anda digunakan. Privasi Anda adalah fondasi kepercayaan kami.

Data yang Dikumpulkan

Kasir Plus menyimpan data transaksi dan produk hanya di perangkat Anda. Tidak ada data pribadi yang dikirim ke server eksternal.

  • Data disimpan secara lokal
  • Tidak ada pengumpulan identitas pribadi
  • Firebase Analytics merekam data penggunaan anonim (ID sesi, halaman dilihat, crash)

Izin Aplikasi

Izin minimum yang diperlukan untuk fungsionalitas inti:

  • Penyimpanan: untuk database lokal transaksi
  • Tidak mengakses kontak
  • Tidak mengakses lokasi
  • Tidak mengakses data pribadi lain

Layanan Pihak Ketiga

Kasir Plus menggunakan layanan berikut untuk meningkatkan pengalaman:

Google AdMob

Menampilkan iklan dalam aplikasi. AdMob dapat menggunakan ID perangkat untuk personalisasi iklan. Anda dapat mengatur ulang ID iklan atau menonaktifkan personalisasi di Pengaturan Google ➔ Iklan.

Firebase Analytics

Mengumpulkan data penggunaan anonim (seperti halaman yang dikunjungi, interaksi tombol, laporan error) untuk membantu kami menyempurnakan aplikasi. Tidak ada informasi pribadi yang direkam.

Keamanan Data

Semua data transaksi Anda tetap berada di perangkat dan tidak dapat diakses oleh pihak luar.

  • Penyimpanan lokal terenkripsi
  • Tidak ada akses jarak jauh
  • Anda pemilik penuh data Anda

Perubahan Kebijakan

Kebijakan ini dapat diperbarui sewaktu-waktu. Versi terbaru selalu tersedia di aplikasi.
