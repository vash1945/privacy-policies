Kebijakan Privasi KasirPlus

Kami menghargai privasi Anda dan berkomitmen untuk melindungi data pribadi pengguna aplikasi kami

Data yang Dikumpulkan

KasirPlus dirancang dengan prinsip privasi sebagai prioritas utama. Aplikasi ini hanya menyimpan data transaksi dan informasi produk secara lokal di perangkat Anda.

Tidak ada data yang dikirim atau disimpan di server pihak ketiga

Tidak ada pengumpulan data pribadi pengguna

Semua data tetap berada di perangkat pengguna

Izin Aplikasi

KasirPlus memerlukan izin terbatas untuk berfungsi dengan baik:

Izin Penyimpanan: Digunakan untuk menyimpan database transaksi dan produk secara lokal di perangkat Anda. Tanpa izin ini, aplikasi tidak dapat menyimpan data transaksi Anda.

Tidak memerlukan akses ke kontak

Tidak memerlukan akses ke lokasi

Tidak memerlukan akses ke data pribadi lainnya

Pihak Ketiga

KasirPlus tidak menggunakan layanan pihak ketiga yang mengumpulkan atau memproses data pengguna.

Tidak ada integrasi dengan platform analitik

Tidak ada integrasi dengan platform iklan

Tidak ada berbagi data dengan entitas eksternal

Keamanan Data

Keamanan data pengguna adalah prioritas utama kami:

Semua data disimpan secara lokal di perangkat Anda

Tidak ada akses eksternal ke data Anda

Data Anda tetap menjadi milik Anda sepenuhnya
